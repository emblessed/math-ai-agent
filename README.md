# 🧮 Math AI Agent (phi4-mini + mathjs)

Интеллектуальный математический помощник на базе локальной LLM (**phi4-mini**), который умеет работать с переменными, сложными вычислениями и матрицами, сохраняя контекст между сообщениями.

## ✨ Особенности
- **Локальный запуск:** Конфиденциальность данных (используется Ollama).
- **Умная память:** Агент запоминает переменные (цена, налоги, x, y) и может использовать их в последующих расчетах.
- **Поддержка кириллицы:** Вы можете называть переменные по-русски (`'цена'`, `'продукт'`).
- **Гибридный режим:** Автоматическое переключение между калькулятором и режимом чата.

## 🛠 Синтаксис управления

Для стабильной работы и разделения логики в агенте реализованы следующие правила:

1. **Создание переменной (Одинарные кавычки):**
   - Используйте `' '` для объявления переменных.
   - Пример: `'цена' = 1500` или `'налог' = 0.13`
2. **Математические операции:**
   - Агент понимает стандартные знаки: `+`, `-`, `*`, `/`, `^` (степень).
   - Пример: `цена * (1 - налог)`
3. **Обсуждение (Двойные кавычки):**
   - Используйте `" "` в обычном текстовом запросе, если хотите спросить что-то о переменной.
   - Пример: `Что ты думаешь про значение "цена"?`
